## Project Description
This project is a Memory Card Game developed using **HTML, CSS, and JavaScript (DOM Manipulation)**.  
The player flips cards to find matching pairs. Matching cards remain face-up, while non-matching cards flip back after a short delay.

The game also includes a **timer, move counter, and scoring system** to track player performance.

## How to Run the Project
1. Download or copy the project files.
2. Open the `index.html` file in any modern web browser.
3. The game will start automatically.

---

## How to Play
1. Click on a card to flip it.
2. Flip two cards at a time.
3. If the cards match, they remain open.
4. If they do not match, they flip back after a short delay.
5. Each pair attempt increases the move counter.
6. Each correct match increases the score.
7. The timer runs while the game is active.
8. Click **Restart Game** to play again.

---

## Game Rules
- Only two cards can be flipped at one time.
- Matched cards cannot be flipped again.
- The game continues until all pairs are matched.
- Restarting the game resets time, moves, and score.
